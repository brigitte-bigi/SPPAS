============================================================================
lang
============================================================================

Linguistic resources to enable the support of a given language in several
automatic annotations of SPPAS including but not limited to TextNormalization,
Phonetization, Alignment and Syllabification.

cuedspeech.zip:
===============

This resource contain the rules to generate the key codes of the French
Cued Speech, LfPC - Langue française Parlée Complétée.
It also contains a set of photos of hands to represent the consonants.
All these resources are:
    * Copyright (C) 2022 Brigitte Bigi, LPL, CNRS.
    * GNU General Public License, version 3

facedetect.zip:
===============

This resource contain a set of models for Haar Cascade Classifier and
Artificial Neural Networks to perform Automatic FaceDetection.
A copyright is mentioned inside each file.


facemark.zip:
===============

This resource contain a set of models to perform Face Landmark.
A copyright is mentioned inside files and the license in a readme
file.
